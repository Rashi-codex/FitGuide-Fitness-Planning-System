# 🏋️ FitGuide – Personal Fitness Assistant

FitGuide is a web-based fitness application developed using Flask that helps users maintain a healthy lifestyle through personalized fitness guidance, workout recommendations, and nutrition planning.

The platform provides useful health calculators and customized fitness suggestions based on user goals, making it easier to plan workouts and manage overall wellness.

---

## 🚀 Features

### 👤 User Fitness Profile
- Create a personalized fitness profile
- Goal-based fitness recommendations
- Activity-level assessment
- Customized health guidance

### 💪 Exercise Recommendations
- Strength training workouts
- Calisthenics exercise plans
- Flexibility and mobility exercises
- Beginner-friendly workout suggestions

### 🥗 Diet & Nutrition Planning
- Weight loss diet plans
- Muscle gain nutrition recommendations
- Balanced diet suggestions
- Healthy eating guidance

### 📊 Health & Fitness Calculators
- BMI (Body Mass Index) Calculator
- Daily Calorie Requirement Calculator
- Protein Intake Calculator

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/FitGuide-Fitness-Planning-System.git
```

### 2. Navigate to Project Directory

```bash
cd FitGuide-Fitness-Planning-System
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```text
http://127.0.0.1:5000
```
